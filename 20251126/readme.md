# ToeicMate 🎯

> 생성형 AI를 활용한 개인 맞춤형 TOEIC 학습 플랫폼

10,000명 이상의 사용자를 위한 AI 기반 토익 학습 앱입니다. 파트별 학습, AI 튜터, 실시간 약점 분석 등 다양한 기능을 제공합니다.

<br>

## ✨ 주요 기능

### 🤖 생성형 AI 활용
- **AI 튜터**: 실시간 질문/답변 및 학습 가이드
- **AI 문제 생성**: 파트별 맞춤 문제 자동 생성
- **약점 분석**: AI 기반 학습 데이터 분석 및 개선 방향 제시

### 📚 학습 기능
- **7개 파트 완벽 대응**: TOEIC Part 1~7 전체 커버
- **파트별 연습 모드**: 각 파트별 집중 학습
- **모의고사 모드**: 실전 시험 대비 (2시간 타이머)
- **리스닝 연습**: 오디오 재생 및 반복 학습

### 📊 학습 관리
- **실시간 점수 추적**: 현재 점수 → 목표 점수 진척도 표시
- **스트릭 시스템**: 연속 학습 동기 부여
- **일일 목표**: 하루 학습 목표 설정 및 달성률 표시
- **상세 대시보드**: 
  - 파트별 정답률 그래프
  - 레이더 차트로 강/약점 시각화
  - 학습 통계 및 히스토리

### 📖 단어장
- **자동 단어 수집**: 틀린 문제의 핵심 단어 자동 저장
- **복습 시스템**: 저장된 단어 반복 학습

### 🏆 게이미피케이션
- **레벨 시스템**: Bronze → Silver → Gold → Platinum
- **배지 획득**: 점수 달성 시 배지 수여
- **포인트 시스템**: 학습 활동에 따른 포인트 적립

### 👤 사용자 관리
- **회원가입/로그인**: 개인 데이터 관리
- **데이터 동기화**: 자동 백업 및 복원
- **프로필 관리**: 목표 점수 및 학습 설정

<br>

## 🛠️ 기술 스택

| 분류 | 기술 |
|------|------|
| **Frontend** | React 18 + Vite |
| **스타일링** | Tailwind CSS |
| **차트** | Recharts |
| **아이콘** | Lucide React |
| **AI API** | 생성형 AI API (문제 생성, 채팅, 분석) |
| **상태 관리** | React Hooks (useState, useEffect, useRef) |
| **데이터 저장** | LocalStorage + Backend Sync |

## 🚀 설치 및 실행

### 1. 필수 요구사항
- Node.js 16.0 이상
- npm 또는 yarn

### 2. 설치
```bash
# 저장소 클론
git clone [repository-url]
cd ToeicMate

# 의존성 설치
npm install
```

### 3. 개발 서버 실행
```bash
npm run dev
```

### 4. 빌드
```bash
npm run build
```
